# Online-Food-Ordering-System-Console-Application-Java-
It's a system that asks the user if he's vegetarian or not. Whether  vegetarian or not, the program will display a menu that contains a list of vegetarian or non-vegetarian dishes and user will choose the delivery method. The system validates user input using custom exceptions and updates the menu dynamically after the user selects a dish.

## Food Categories

Vegetarian

Non-Vegetarian

## Menu Management

1.Adding dishes to a category

2.Displaying menu items

3.Removing a selected dish from the menu

4.Validating dish names using custom exceptions

5.Managing a static menu list (vegetarianList)

## Delivery Options

1.BikeDelivery

2.CarDelivery

3.DroneDelivery
